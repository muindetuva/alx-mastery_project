# Requirements

## Timer

- As a user, I want to start a 25-minute timer so that I can work in a structured focus block.
- As a user, I want to pause the timer so that I can handle short interruptions without losing the session.
- As a user, I want to resume a paused timer so that I can continue the same focus block.
- As a user, I want to reset the timer so that I can restart a session if I chose the wrong task.
- As a user, I want to see the remaining time clearly so that I always know how much focus time is left.

## Tasks

- As a user, I want to name what I am working on so that each focus session has a clear purpose.
- As a user, I want to add tasks to a task list so that I can plan multiple work items.
- As a user, I want to mark a task as complete so that I can track progress.
- As a user, I want to select a task before starting the timer so that completed sessions are connected to real work.
- As a user, I want to delete a task so that old or mistaken tasks do not clutter my list.

## Breaks

- As a user, I want to be prompted to take a break when the timer ends so that I recover before the next focus block.
- As a user, I want to start a short break timer so that my work rhythm stays consistent.
- As a user, I want to skip a break so that I can continue working when necessary.

## Session Log

- As a user, I want completed focus sessions to be saved so that I can review my work history.
- As a user, I want each session to record the task name, duration, and completion time so that the log is useful later.
- As a user, I want abandoned sessions to be excluded from completed totals so that my statistics stay accurate.
- As a user, I want data to persist after a browser refresh so that I do not lose tasks or session history.

## Insights

- As a user, I want to see total focused time so that I can understand how much deep work I completed.
- As a user, I want to see which tasks took the longest so that I can identify demanding work.
- As a user, I want to see daily focus totals so that I can notice work patterns over time.

## Settings

- As a user, I want the default focus duration to be 25 minutes so that the app follows a familiar Pomodoro pattern.
- As a user, I want break duration to have a sensible default so that I can use the app without configuration.
- As a user, I want simple controls that work on mobile and desktop so that I can use the timer anywhere.
