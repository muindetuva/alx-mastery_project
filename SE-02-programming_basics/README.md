# SE-02 Programming Basics

This project is a command-line quiz engine built in Python.

## Current Files

- `prompt-log.md`: planning notes and question bank prompt log
- `quiz.py`: quiz loop logic, input validation, asked-question tracking, and score updates

## Planned Modules

- `question_bank.py`: stores quiz questions
- `quiz.py`: runs the category quiz loop
- `results.py`: displays final scores and performance messages
- `main.py`: connects the modules and starts the program
